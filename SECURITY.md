# Security Policy

## Supported Versions

All versions of this software are currently a work in progress, and no security updates will be provided untill the software actually works.

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, simply open an Issue on this Github page. If we can't replicate the flaw, we will ask for more information, but otherwise, we will do the rest!
